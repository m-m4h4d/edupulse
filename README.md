# EduPulse

Advanced E-Learning App made in Flutter/Dart.

## Features

This app has following feature:

- User Authentication
- Course Management
- Quiz Management
- Assignment Management
- Notification Management
- Profile Management
- Course Progress Tracking
- Quiz Results Tracking
- Assignment Results Tracking
- Notification History Tracking
- Profile Settings
- Course Settings
- Quiz Settings
- Assignment Settings
- Notification Settings

## Installation

```bash
flutter create edupulse
```

## Running the App

```bash
flutter run
```

## Running the Tests

```bash
flutter test
```

## Running the Widget Tests

```bash
flutter test test/widget_test.dart
```

## Running the Integration Tests

```bash
flutter test test/integration_test.dart
```

## Running the All Tests

flutter test
